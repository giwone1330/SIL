## Hotkeys
- Build / Compile from tex
  - ctrl + alt + b
  - or just save file
- View PDF
  - ctrl + alt + v
- Code to PDF
  - ctrl + alt + j
- PDF to Code
  - ctrl + click
- Snippets
  - [cheatsheet](https://cheatography.com/jcwinkler/cheat-sheets/latex-workshop-visual-studio-code/)
  - study how to make user snippets later...
